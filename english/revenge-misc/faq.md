#### How do I install a plugin?
Copy the link, and then:
1. Go to Settings
2. Search for the Revenge section
3. Click on Plugins
4. Click on '+' button
5. Click on 'import from clipboard' button
6. Click on 'install' button
> [!WARNING] 
> Keep in mind that this links are unproxied. In order to have more safe plugins install 'Plugin browser' following the previous steps and then install all plugins from there.


#### Where can I find plugins to install?
You can search for them in the [community maintained list](https://purple-eyez.github.io/Plugins-List/).

#### I want 'x' plugin!
Search for it on the community maintained list, on plugin browser or just use a search engine. If you can not find it using any of those methods, it's likely that it doesn't exist.

#### Why can't I enable some plugins?
They are broken. Either look for an alternative or if there isn't make your own, or cry about it.

#### How do I use my own personal font?
Read the .json files of the FAQ avobe and make your own. You can host it freely in github like these are. If you have more knowledge you could do it in a private server if you don't want to make it public.

#### How do I install those fonts?
Copy the link, and then:
1. Go to Settings
2. Search for the Revenge section
3. Click on Fonts
4. Click on 'Install a font' button
5. Click on 'Import font entries from a link'
6. Paste the link
7. Click on 'Import button' button
  You could import it manually without using a JSON but this is way faster and assures you won't misconfig anything.
