# Revenge via Xposed Framework

Welcome! This is an unofficial guide on how to install Revenge using the Xposed framework. This method is recommended for advanced users who have experience with Android modifications.

## Overview

The Xposed framework method allows you to inject Revenge into Discord without modifying the APK directly. This approach offers several advantages:

- ✅ **No need to re-patch** after Discord updates
- ✅ **Cleaner installation** process
- ✅ **Better compatibility** with other Xposed modules
- ⚠️ **Requires root access** and technical knowledge

The installation consists of two main steps:
