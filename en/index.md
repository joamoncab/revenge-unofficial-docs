# English Documentation

Welcome to the English documentation for Revenge modded client! This comprehensive guide will help you install, configure, and use Revenge effectively.

## What is Revenge?

Revenge is a modded Discord client that provides additional features and customization options beyond the standard Discord experience. This documentation covers various installation methods and usage tips.
