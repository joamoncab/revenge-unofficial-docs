## Prerequisites
In order to successfully install Revenge via xposed, you will need the following:
1. Discord installed, make sure it's the latest version.
2. An android device with unlocked bootloader
3. Root access via [Magisk](https://github.com/topjohnwu/Magisk) or [KernelSu](https://github.com/tiann/KernelSU)/[KernelSuNext](https://github.com/KernelSU-Next/KernelSU-Next)
4. zygisk, either via:
	1. Built-in or [ReZygisk](https://github.com/PerformanC/ReZygisk) for [Magisk](https://github.com/topjohnwu/Magisk)
	2. [ReZygisk](https://github.com/PerformanC/ReZygisk) for [KernelSu](https://github.com/tiann/KernelSU)/[KernelSuNext](https://github.com/KernelSU-Next/KernelSU-Next)
5. A working LSposed installation such as the [Jing Matrix](https://github.com/JingMatrix/LSPosed) fork.
> Since ZygiskNext is not open source, it is not recommended
