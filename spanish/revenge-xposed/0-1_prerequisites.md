## Prerrequisitos
Para instalar satisfactoriamente Revenge con xposed, necesitarás lo siguiente:
1. Discord instalado, asegúrate de que es la última versión.
2. Dispositivo Android con bootloader desbloqueado
3. Root mediante [Magisk](https://github.com/topjohnwu/Magisk) o [KernelSu](https://github.com/tiann/KernelSU)/[KernelSuNext](https://github.com/KernelSU-Next/KernelSU-Next)
4. zygisk, tanto a través de:
	1. integrado o [ReZygisk](https://github.com/PerformanC/ReZygisk) en [Magisk](https://github.com/topjohnwu/Magisk)
	2. [ReZygisk](https://github.com/PerformanC/ReZygisk) para [KernelSu](https://github.com/tiann/KernelSU)/[KernelSuNext](https://github.com/KernelSU-Next/KernelSU-Next)
5. Una instalación funcional de LSPosed como el fork de [Jing Matrix](https://github.com/JingMatrix/LSPosed).
> Como ZygiskNext no es de código abierto, no es recomendable.
