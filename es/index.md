# Documentación en Español

¡Bienvenido a la documentación en español para el cliente modificado Revenge! Esta guía completa te ayudará a instalar, configurar y usar Revenge de manera efectiva.

## ¿Qué es Revenge?

Revenge es un cliente modificado de Discord que proporciona funciones adicionales y opciones de personalización más allá de la experiencia estándar de Discord. Esta documentación cubre varios métodos de instalación y consejos de uso.
