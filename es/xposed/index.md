# Revenge vía Framework Xposed

¡Bienvenido! Esta es una guía no oficial sobre cómo instalar Revenge usando el framework Xposed. Este método está recomendado para usuarios avanzados que tienen experiencia con modificaciones de Android.

## Resumen

El método del framework Xposed te permite inyectar Revenge en Discord sin modificar directamente el APK. Este enfoque ofrece varias ventajas:

- ✅ **No necesitas re-parchear** después de actualizaciones de Discord
- ✅ **Proceso de instalación más limpio**
- ✅ **Mejor compatibilidad** con otros módulos Xposed
- ⚠️ **Requiere acceso root** y conocimientos técnicos

## Proceso de Instalación

La instalación consiste en dos pasos principales:

<div class="vp-doc">

### Paso 1: Prerrequisitos
**[Verificar Prerrequisitos →](/es/xposed/prerequisites)**

Antes de comenzar, asegúrate de tener todos los componentes requeridos:
- Dispositivo Android rooteado con bootloader desbloqueado
- Framework Xposed (LSPosed recomendado)
- Última versión de Discord
- Comprensión básica de modificaciones Android

### Paso 2: Instalación
**[Instalar Revenge →](/es/xposed/install)**

Sigue la guía detallada de instalación con capturas de pantalla:
- Descargar el módulo Revenge-Xposed
- Instalar y configurar en LSPosed
- Habilitar para la aplicación Discord
- Verificar la instalación
